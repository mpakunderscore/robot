Folker

Autonomous Terrain & Security Quadruped
Folker is a field-optimized quadruped robot designed for long-term autonomous deployment in harsh natural environments. With a carbon-fiber polymer frame and sealed joints, it operates independently in low-connectivity wilderness with rugged terrain.
Environmental Intelligence
Tracks temperature, humidity, terrain topology, and kinetic activity through onboard multi-modal sensors.
Security & Patrol AI
Operates real-time patrol routes using AI-assisted SLAM and thermal/motion vision to detect intrusion or anomalies.
Autonomy at Core
No remote operator needed. Powered by onboard compute, obstacle detection, and adaptive terrain routing.
Mesh Communication
Connects with other Folker units using decentralized LoRa-based mesh — enabling data relay in blackout zones.
Solar-Powered Missions
Equipped with efficient solar charging panels for continuous multi-day operation with idle recharge capability.
Swarm Terrain Coverage
Multiple units dynamically link into a self-healing mesh swarm, syncing patrols and sharing environment data.
Differentiators
Folker is built from the ground up for remote land autonomy, prioritizing cost-efficiency, off-grid mesh networking, and survivability over general-purpose robotics performance.
Specifications
Weight: 3.1 kg
Top Speed: 1.2 m/s
Battery Runtime: 3h active, 12+ hours standby with solar input
Sensors: RGB Stereo, IMU, Thermal, LiDAR
Communication: LoRa, WiFi, Bluetooth
Parts
Core: Raspberry Pi 5 (8GB) + active cooling
AI HAT: Raspberry Pi AI HAT+ with Hailo-8 accelerator
Motors x12: GIM4305 with integrated CAN drivers
CAN: Waveshare CAN FD HAT (SPI)
Co-Processor: ESP32-S3 (BLE, watchdog, safety fallback)
PSU: 44.4V → 12V 20A + 5V 5A buck converters
BMS: JBD Smart BMS 12S-80A (with UART/CAN)
Battery: 18650 x24 (12S2P, 3000/6000mAh cells = 266/532 Wh)
Sensors: MPU-9250 + BMP280 (SPI, vibration-isolated)
LoRa: RAK19007 + RAK4631 (LoRa + BLE)
LTE: Waveshare SIM7600E-H 4G (cellular + GNSS)
Camera: StereoPi v2 + 2x IMX219 160° fisheye
Cooling: Fans for Pi + PSU and optional motor cooling
Chassis: Modular, dampened, power-isolated layout
